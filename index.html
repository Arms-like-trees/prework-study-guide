Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • While the head elements is a container for metadata. Meta elements contain information about the page that is used by the browser. The user won't see information in the head or meta elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Separation of concerns is a widely used term in web development and refers to serparting code into different files based on use. It is one of the key design principles that we practice in Boot Camp. Separation of concerns is based on the idea that each section of code should have its own responsibility
  • The order of CSS from inner to outer is as follows: Content, padding, border, margin
  • Do not repeats yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained. DRY can greatly reduce their overall size and complexity of a code base
  • Debugging is one of the most important parts of the development process. Every day, developers notice errors and review code that isn’t working. It’s important to get into the mindset that errors and mistakes are gifts. They tell us when something is wrong, and if we can stay curious, we can figure out what is happening and learn something new.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a branch and switches to it
  • git checkout main: switches to the main branch
  • A pull request, or PR, is used to merge a comparison branch( or featured branch) into the base branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Contol flow is the order in which a computer executes code in a script
  • JAvaScript is a powerful programmin language that can add interactivity to a website
  • Some different types of JAvaScript are:Null, Undefined, Booleon, Number, BigInt, String, and Symbol.
  • Null is inhabited by exactly one value: null
  • Undefined is inhabited by exactly one value: undefined.
  • Boolean represents logical entity and is inhabited by true and false
  • Number is a double prediction 64-bit binary format IEEE 754 value
  • BigInt is a numberic primitive that can represent intergers with arbitrary magnitude. With BigInts you can safetly store and operate on large interger even beyond the interger limit for Numbers
  • String represents textual data and is encoded as a sequence of 16-bit unsigned interference values representing UTF-16 code units
  • Symbol is a unique and immutable primitive value and may be used as the key of an object property
